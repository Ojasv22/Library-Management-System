# Library Management System (Java)

## Project Overview
This is a simple console-based Library Management System developed using Java. 
The project helps manage books in a library by allowing the user to add, issue, return, and display books.

## OOP Concepts Used
- Classes and Objects
- Inheritance
- Interface
- Packages
- Exception Handling

## Class Description
- Book: Represents a book with ID, title, and status.
- LibraryOperations: Interface defining library operations.
- Library: Implements library operations and manages books.
- Main: Contains the main method and menu-driven logic.

## How to Compile and Run
1. Open terminal/command prompt.
2. Navigate to src folder.
3. Compile:
   javac library/*.java
4. Run:
   java library.Main

## Output
The program displays a menu and performs operations based on user input.
